# Cassie Marie — GitHub Pages Site

This package has been cleaned up for plain GitHub Pages hosting.

## Centralized site-shell controls
- `/js/site-config.js` = main nav labels, footer labels, social labels, copyright copy
- `/js/site-shell.js` = applies the shared shell across pages
- `/css/style.css` = includes the all-caps override rules
